# Web版マイノリティゲーム
大人数でマイノリティゲームを行うためだけのサイト
## 動作方法
### ローカルで動かす場合
1. プロジェクトをクローンします
2. npm install が出来ていなかったら行ってください
   > npm install
3. npm startでアプリを起動
   > npm start
4. http://localhost:3000 にアクセス
5. 管理者の初期パス"admin252"でログイン

### 公開する場合
1. 自身のGitHubのプリジェクトにこのプロジェクトをアップする
2. https://render.com/ で無料の枠でGitHubと連携してアップしたプロジェクトを動かす
3. 発行されたURLを共有
4. 管理者の初期パス"admin252"でログイン


## ゲームの進行
### 管理者
https://minority-game.onrender.com/admin.html
1. 管理者パスワード、参加者パスワードを変更することが可能
2. クイズは事前に作成してキューに入れておくことで進めることが可能
3. 投票が終わったら結果表示ボタンを押す。すると結果が参加者に反映される
4. 繰り返して、任意のタイミングで最終結果発表を行うことで勝者が確定する
5. 全部終わったらリセットを実施

### 参加者
https://minority-game.onrender.com/index.html
1. ログイン後、任意で表示名を変更できる
2. 管理者が問題を発行すると投票が行えるようになる
3. 結果表示までは変更可能
4. 投票中は他の参加者の投票状況は分からない
5. 結果が表示され、マイノリティ(少数派)なら「参加中」表示のままになって、マジョリティ(多数派)なら脱落で以降の問題に投票出来なくなる
6. 未投票の場合も脱落扱いとなる
7. 何戦か繰り返し、最終結果発表して残っていた人はWinnerと表示される

### ディスプレイ
https://minority-game.onrender.com/display.html
1. ログイン無しでアクセス可能
2. 現在の問題、試合状況などの表示用
3. 最終結果発表を行うと勝者を表示する
